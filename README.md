# customer-sales-analysis-sql
SQL project for customer and sales data analysis

📊 Customer & Sales Analysis (SQL Project)
🎯 Obiettivo

Analizzare i dati di vendite e clienti per individuare i clienti più profittevoli, i prodotti con maggiore impatto sul fatturato e i principali trend di vendita.

📁 Dataset
Il dataset simula un contesto aziendale (retail/servizi) ed è composto da:
customers: dati dei clienti
orders: ordini effettuati e valore totale
order_items: dettaglio dei prodotti acquistati


🛠️ Strumenti utilizzati
MySQL (MySQL Workbench)


❓ Domande di business
Quali sono i clienti che generano più fatturato?
Quali prodotti contribuiscono maggiormente ai ricavi?
Come variano le vendite nel tempo?
Qual è la spesa media per cliente?


🔍 Analisi svolte
Identificazione dei top clienti per spesa
Analisi del fatturato totale e medio
Calcolo del contributo dei prodotti al fatturato
Analisi delle vendite per periodo


💡 Insight principali
Una parte ridotta di clienti genera una quota significativa del fatturato
Alcuni prodotti hanno un impatto maggiore sui ricavi totali
Le vendite mostrano variazioni nel tempo utili per analisi stagionali
I clienti presentano comportamenti di spesa diversi


📈 Conclusioni
L’analisi dimostra come l’utilizzo di SQL permetta di trasformare i dati in informazioni utili per supportare decisioni aziendali.
